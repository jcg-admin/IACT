# Playbooks operativos

Este directorio agrupa tareas repetibles para soporte y operaciones. Cada archivo documenta un procedimiento específico y se acompaña de checklists de validación.

## Convenciones
- Un archivo por procedimiento (`*.md`).
- Títulos descriptivos que incluyan la plataforma o servicio afectado.
- Incluir contexto, pasos automatizados, validaciones manuales y rutas de evidencia.
- Referenciar scripts o infraestructura dentro del repositorio con rutas relativas.
