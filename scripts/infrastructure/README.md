# Infrastructure Scripts

Scripts de infraestructura, CI/CD, monitoring y operaciones.

## Directorios

- `cassandra/` - Scripts para base de datos Cassandra
- `ci/` - Continuous Integration / Continuous Deployment
- `logging/` - Configuración y gestión de logs
- `disaster_recovery/` - Scripts de recuperación ante desastres
- `load_testing/` - Scripts de pruebas de carga
- `benchmarking/` - Scripts de benchmarking
- `dev/` - Herramientas de desarrollo

## Estructura

Cada subdirectorio contiene su propio README con documentación específica.

## Referencias

Ver documentación en cada subdirectorio para uso detallado.
