# Workflow Management

Scripts para generación y validación de workflows.

## Scripts

- `generate_from_template.py` - Genera workflows desde plantillas
- `check_no_emojis.py` - Valida que no haya emojis en archivos

## Uso

```bash
# Generar workflow desde plantilla
python scripts/workflows/generate_from_template.py --template TEMPLATE_NAME

# Validar emojis
python scripts/workflows/check_no_emojis.py
```
