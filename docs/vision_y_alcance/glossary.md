| Término | Definición breve |
|---------|------------------|
| IVR | Respuesta de voz interactiva usada para enrutar llamadas entrantes. |
| RN | Requisito no funcional que describe restricciones de calidad o plataforma. |
| RQ | Requisito funcional numerado bajo el esquema RQ-ANL-###. |
| UC | Caso de uso con numeración UC-DASH-### que refleja escenarios clave. |
| TC | Caso de prueba registrado como TC-USR-### para validar cada UC. |
| ADR | Architecture Decision Record documentado como ADR-AAAA-###. |
| SLA | Acuerdo de nivel de servicio para tiempos de respuesta y disponibilidad. |
| KPI | Indicador clave para medir desempeño del call center y sus agentes. |
| ETL | Proceso de extraer, transformar y cargar datos hacia el data warehouse. |
| BI | Capa de analítica de negocio que consume los resultados de la ETL. |
| RBAC | Control de acceso basado en roles aplicado a dashboards y APIs. |
| DR | Estrategia de recuperación ante desastres para restaurar servicios críticos. |
| MTTR | Tiempo medio de recuperación usado en reportes operativos. |
| NPS | Métrica de satisfacción del cliente recopilada desde encuestas IVR. |
| APScheduler | Planificador empleado para tareas periódicas dentro del monolito. |
