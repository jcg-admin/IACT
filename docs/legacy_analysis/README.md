## Por qué están archivados

Estos documentos fueron creados durante el proceso de análisis y reestructuración de la documentación del proyecto. Aunque fueron muy útiles en su momento para:

1. Evaluar diferentes enfoques de organización documental
2. Comparar frameworks (BABOK vs PMBOK)
3. Iterar sobre la estructura óptima
4. Documentar el proceso de decisión

Ahora representan:
- **Redundancia**: El contenido útil está consolidado en `PROPUESTA_FINAL_REESTRUCTURACION.md`
- **Confusión potencial**: Múltiples versiones pueden generar duda sobre cuál es la actual
- **Espacio**: ~168KB de documentación redundante
