# Plantillas de Colaboración - Backend

Las plantillas de GitHub Issues y PRs están en `.github/`:
- `.github/ISSUE_TEMPLATE/` - Templates de issues
- `.github/PULL_REQUEST_TEMPLATE.md` - Template de PRs
