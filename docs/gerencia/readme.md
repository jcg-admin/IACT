# Espacio de documentación - Gestión y visión

Este espacio centraliza toda la información de carácter gerencial: visión del producto, objetivos estratégicos, métricas de éxito y dependencias con otras áreas. A partir de ahora cualquier iniciativa relacionada con estrategia o alineación ejecutiva debe registrarse aquí.

## Página padre
- [`../index.md`](../index.md)

## Subdirectorios

- [`vision_y_alcance/`](vision_y_alcance/readme.md): describe la propuesta de valor, objetivos estratégicos y backlog de visión.
- [`vision_y_alcance/glossary.md`](vision_y_alcance/glossary.md): glosario oficial para uniformar vocabulario entre negocio y tecnología.

## Lineamientos

1. Toda nueva decisión estratégica debe anotarse en `docs/registro_decisiones.md` y, si corresponde, ampliarse en esta sección.
2. Documentos operativos o tácticos (por ejemplo, planificación detallada o gobierno) deben enlazarse desde este índice para mantener trazabilidad.
3. Evita mezclar artefactos técnicos (backend/frontend) dentro de esta jerarquía; usa los espacios dedicados para cada dominio.

> Mantén este índice actualizado para que dirección y equipo técnico encuentren rápidamente los artefactos de alto nivel.
