# Requerimientos de Negocio (Nivel 2)

Marco conceptual: `docs/gobernanza/marco_integrado/marco_reglas_negocio.md`

## Descripción

Los requerimientos de negocio describen los objetivos organizacionales y las necesidades que el sistema de IA debe satisfacer.

## Tipos de Documentos

- Objetivos estratégicos relacionados con IA
- Necesidades del negocio
- Alcance del sistema
- Restricciones del proyecto

## Trazabilidad

Nivel 2 alimenta: Nivel 3 (Requerimientos de Usuario)
