# Casos de Uso - Especificaciones

## Nomenclatura

`UC-NNN-{verbo_objeto}.md`

Ejemplo: `UC-001-registrar_usuario.md`

## Formato

Cada especificación debe usar formato completo de dos columnas:

| ACCIONES DEL ACTOR | RESPONSABILIDADES DEL SISTEMA |
|-------------------|------------------------------|

## Estructura Completa

Ver plantilla en: `docs/gobernanza/marco_integrado/marco_casos_uso.md`
