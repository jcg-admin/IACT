## Razón de la Migración

Las metodologías y procesos pertenecen naturalmente a la **estructura de gobernanza**, no a documentación técnica de desarrollo. Esta migración:

✅ **Mejora la organización**: Metodologías en un lugar centralizado
✅ **Elimina duplicación**: Versiones únicas y actualizadas
✅ **Facilita mantenimiento**: Un solo lugar para actualizar
✅ **Sigue estándares**: Gobernanza contiene metodologías y procesos
