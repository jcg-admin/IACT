# 01 · Gobernanza

Define el marco de decisión y coordinación que sostiene el pipeline de documentación. Aquí se asignan responsables y cadencias
para que las sesiones capturadas con la plantilla de actividades se traduzcan en acuerdos accionables.

## Rol dentro del pipeline
- **Paso 3. Documentar logística.** Calendarios, foros y enlaces se centralizan aquí.
- **Paso 4. Listar asistentes y roles.** Mantiene la matriz RACI y los comités activos.
- **Paso 7. Action items.** Consolida el backlog operativo y hace seguimiento a compromisos.

## Artefactos obligatorios
- Lineamientos vigentes (`lineamientos_gobernanza.md`).
- Matriz RACI (pendiente, basarse en `docs/plantillas/plantilla_stakeholder_analysis.md`).
- Calendario de rituales con enlaces a registros (`plantilla_registro_actividad.md`).
