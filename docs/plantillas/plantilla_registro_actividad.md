# Plantilla Registro de Actividades o Planeaciones

Esta plantilla estandariza la captura de notas de reunión, planes de trabajo o seguimientos de actividades.
Completa cada sección reemplazando los valores entre `<>` y eliminando los comentarios de ayuda.

```

Creado por <Nombre del autor>, actualizado por última vez el <fecha en formato corto> <tiempo estimado de lectura>
```
