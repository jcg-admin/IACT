# Anexos

Repositorio de materiales complementarios: glosarios extendidos, referencias externas y ejemplos detallados. Todos los anexos deben enlazarse desde la sección correspondiente para mantener trazabilidad.

## Subdirectorios
- `diagramas/`: diagramas de alto nivel utilizados en sesiones.
- `ejemplos/`: casos ilustrativos y walkthroughs.
- `referencias/`: enlaces y documentación corporativa.

## Buenas prácticas
- Documentar propietario y fecha de actualización en cada anexo.
- Mantener consistencia con la taxonomía definida en [`../documentacion_corporativa.md`](../documentacion_corporativa.md).
- Referenciar anexos críticos desde [`../planificacion_y_releases/readme.md`](../planificacion_y_releases/readme.md) o el espacio correspondiente.
