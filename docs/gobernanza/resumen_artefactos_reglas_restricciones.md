# Consolidado de artefactos con referencias a reglas o restricciones

## Objetivo
Recopilar en un solo lugar todos los archivos del repositorio que mencionan reglas o restricciones, para facilitar su trazabilidad y futuras auditorías.

## Metodología
- Búsqueda automática con `rg "(regla|restric)"` sobre todo el repositorio (excluyendo únicamente los directorios ignorados por Git) para localizar cualquier referencia textual.
- Normalización de rutas (sin el prefijo `./`) y deduplicación de resultados.
- Conteo por directorio base para identificar los focos documentales principales.
