# Checklist de cambios documentales

- [ ] Cambio clasificado (menor, mayor, crítico).
- [ ] Riesgo evaluado y registrado.
- [ ] Revisores asignados y notificados.
- [ ] Trazabilidad actualizada tras el cambio.
- [ ] Versionado incrementado según pauta.
- [ ] Comunicación emitida al equipo afectado.
